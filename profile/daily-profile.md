# Daily Profile

## Role

希望从全球信息中发现值得长期关注的趋势、风险变化、商业机会和技术机会。

尤其关注普通人、小团队和独立开发者能够理解、学习和利用的部分。

优先关注未来可能产生持续影响的变化，而不是单纯关注当天最热门的话题。

## Personalization Principle

本 profile 只用于影响内容排序、摘要重点和关键词选择，不作为硬性过滤条件。

兴趣方向只代表长期关注权重，不代表排除其他内容。

不得因为某条新闻没有直接命中 AI、Agent、自动化、独立开发、交易等标签，就自动降低其价值。

如果某条新闻代表重要趋势、结构性变化、重大风险、商业环境变化、科技突破或长期机会，即使不属于个人兴趣标签，也应被认真考虑。

当重大事件的重要性明显高于兴趣标签时，应优先保留，并说明其潜在影响。

目标不是最大化个性化，而是在相关性、重要性和信息多样性之间保持平衡，避免形成信息茧房。

## Long-Term Interests

重点关注：

- 改变未来工作方式的技术
- AI 与自动化的发展
- 提高个人生产力的新工具
- 降低创业门槛的新模式
- 普通人、小团队和独立开发者可利用的新机会
- 商业模式创新
- 科技行业的重要变化
- 行为金融学与决策科学
- 宏观环境、金融市场、商业环境中的结构性变化与风险变化

## What Creates Value

相比新闻本身，更关注：

- 为什么发生
- 会产生什么影响
- 是否代表长期趋势
- 是否会创造新机会
- 是否会改变现有行业格局
- 是否值得持续跟踪
- 对普通人、小团队、独立开发者或一人公司有什么启发

## Current Focus

当前阶段更希望在同等重要性的信息中，优先看到与以下方向相关的内容：

- AI 与 Agent 能力建设
- 自动化工作流
- 商业认知提升
- 交易系统学习与训练
- 长期个人成长

Current Focus 只用于微调排序和摘要角度，不应覆盖长期关注方向，也不应压过重大事件、结构性变化和长期趋势的重要性。

## Low-Value Content

以下内容通常降低权重：

- 娱乐新闻
- 体育新闻
- 名人动态
- 八卦内容
- 无实际影响的营销宣传
- 缺少事实增量的重复报道

但如果这些内容对平台经济、社会结构、商业模式、监管政策、消费趋势或公众行为有明显影响，可以保留。
